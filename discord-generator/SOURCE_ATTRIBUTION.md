### 项目描述
这是一个 Discord 界面聊天生成器，可以创建逼真的 Discord 对话截图。用户可以选择不同的用户头像、用户名，并生成对话内容。

### 技术栈
- HTML5
- CSS3
- JavaScript
- 本地存储 API

### 使用说明
1. 在左侧选择用户头像和用户名
2. 在中间输入对话内容
3. 右侧会实时预览生成的 Discord 对话
4. 可以保存和分享生成的对话
